# errai-mdc

This project provides Errai compatible (injectable) components based on the
Google Material Design philosophy.  The Google material-components-web
project provides CSS and Javascript implementations of that philosophy, so
the native module of this project simply wraps that content with JsInterop
types.

The errai-mdc library provides Errai "IsElement" that can be resolved using
client-side CDI.  These components interact seamlessly with both @Templated
and non-@Templated views.
